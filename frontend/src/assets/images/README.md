# Logo Files

Please place your logo files here:

1. **logo-light.png** - Logo for Light Mode
2. **logo-dark.png** - Logo for Dark Mode

## Supported Formats:
- PNG (recommended)
- SVG (recommended for scalability)
- JPG/JPEG

## Recommended Sizes:
- Height: 32-40px (will be scaled to 32px in header)
- Width: Auto (maintain aspect ratio)
- Format: PNG with transparency (for best results)

## File Naming:
- Light mode logo: `logo-light.png` (or `.svg`, `.jpg`)
- Dark mode logo: `logo-dark.png` (or `.svg`, `.jpg`)

The header will automatically switch between logos based on the current theme.
